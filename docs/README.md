# OctoAcme Project Management Documentation Index

## Overview
OctoAcme uses a lightweight, iterative project management approach focused on customer value, clear ownership, and data-informed decisions. Projects move through initiation, planning, execution, risk management, release, and retrospective improvement, with shared artifacts and regular communication to keep cross-functional teams aligned.
